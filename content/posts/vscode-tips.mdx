# VS Code 생산성 200% 높이는 필수 단축키와 설정

VS Code는 가장 인기 있는 코드 에디터입니다. 하지만 기본 설정만으로는 생산성을 극대화하기 어렵습니다.

## 필수 단축키

### 파일 탐색
- `Cmd + P`: 빠른 파일 열기
- `Cmd + Shift + P`: 명령 팔레트
- `Cmd + B`: 사이드바 토글

### 편집
- `Cmd + D`: 같은 단어 선택
- `Alt + Up/Down`: 줄 이동
- `Cmd + Shift + K`: 줄 삭제

## 추천 확장 프로그램

1. **ESLint**: 코드 품질 검사
2. **Prettier**: 자동 포매팅
3. **GitLens**: Git 히스토리 시각화
4. **Thunder Client**: API 테스트

## 설정 팁

```json
{
  "editor.fontSize": 14,
  "editor.minimap.enabled": false,
  "editor.wordWrap": "on",
  "editor.bracketPairColorization.enabled": true
}
```

이 설정들을 적용하면 더 쾌적한 개발 환경을 만들 수 있습니다.
